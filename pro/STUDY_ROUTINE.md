# React 섹션형 학습 루틴 (업데이트: 2026-04-17)

## 운영 방식
- 시간 날 때마다 섹션 1개씩 진행한다.
- 1섹션 권장 시간: 10~15분.
- 한 세션이 끝나면 체크박스 갱신 + 한 줄 메모를 남긴다.
- 명령어 `!다음` 입력 시, 다음 섹션으로 진행한다.
